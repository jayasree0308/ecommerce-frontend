# E-Commerce Website Project

## Overview
This is a **fully functional e-commerce website** built with HTML, CSS, JavaScript, and Firebase. It allows users to browse products, view product details, add items to the cart, sign up/login with Firebase Authentication, and proceed to checkout.  

The website is **responsive, fast, and optimized** for performance, using features such as lazy-loaded images, WebP format, and caching.

---

## Features

### **Homepage**
- Responsive header with **logo, navigation menu, search bar, and cart icon**
- Hero section with **background image, main heading, subheading, and CTA button**
- Optimized images using WebP and lazy loading
- Links to product grid and login/signup pages

### **Product Grid**
- Displays products dynamically from **API or local JSON**
- Each product shows **image, title, price, and Add to Cart button**
- Responsive design: 4 columns on desktop, 2 on tablets, 1 on mobile

### **Product Details Page**
- Displays **full product details** including image, name, price, description
- Users can select **sizes (S, M, L)** and **colors (white, black, blue)**
- Quantity selector with real-time price update
- Add to Cart functionality with storage in **localStorage**

### **Shopping Cart**
- Displays all selected products with quantity, options, and subtotal
- Update quantities or remove items dynamically
- Shows total price and **Proceed to Checkout** button
- Responsive and mobile-friendly

### **Authentication**
- **Signup and login** using Firebase Authentication
- Maintains session persistence across pages
- Logout functionality
- Input validation (email format, password strength)

### **Checkout Page**
- Users can enter **shipping info** and **payment details**
- Shows **order summary** with total amount
- Responsive form layout for desktop and mobile

### **Performance & Optimization**
- Lazy loading of images
- Optimized images in WebP format
- CSS and JS minification
- Caching of static assets (images, CSS, JS)
- Responsive design for all devices

---

## Technologies Used
- **Frontend:** HTML, CSS, JavaScript  
- **Authentication & Hosting:** Firebase (Authentication + Hosting)  
- **Optimization:** WebP images, lazy loading, asset caching  
- **Version Control:** GitHub
